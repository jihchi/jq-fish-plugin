## Prerequisities

- [jq](https://stedolan.github.io/jq)
- [fzf](https://github.com/junegunn/fzf)

## Install

```fish
$ omf install https://github.com/jihchi/jq-fish-plugin.git
```


## Usage

```fish
# read from a file
$ jqrepl [file]

# read from stdin
$ cat <json> | jqrepl
```

## Example

```fish
$ curl -s pie.dev/json | jqrepl
```
